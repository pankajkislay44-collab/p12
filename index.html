Setup Guide

1. Folder Structure

Place your work in the corresponding folders under assets/work/:

  • • logo-design/
  • • package-design/
  • • illustration/
  • • branding/
  • • photography/
  • • graphic-design/

2. Naming Structure

Name your files numerically to match the project data (e.g., 1.jpg, 2.png).

Supported formats: .jpg, .png, .webp

3. Updating Projects

To add or change projects, edit the projects array in the script section at the bottom of index.html.
